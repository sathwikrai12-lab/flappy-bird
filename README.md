## 🎮 About The Game

**Neon Orb** is a fast-paced arcade browser game where you guide a 
glowing crystal ball through endless pipe obstacles. Collect coins, 
grab power-ups, and beat your high score!

Built with pure **HTML5 Canvas + JavaScript** — no libraries, 
no frameworks, just clean code.

---

## ✨ Features

- 🔮 **6 Ball Skins** — Sapphire, Rose, Emerald, Gold, Amethyst, Ember
- 🌙 **Day & Night Theme** — changes automatically every 30 pipes
- ⚡ **3 Power-ups** — Shield, Magnet, Slow-Motion
- 🏆 **High Score Tracker** — saves your best score locally
- 💰 **Coin System** — collect coins for bonus points
- 🔥 **Combo Multiplier** — chain scores for bigger combos
- 📱 **Mobile & Desktop** — works on any device, any screen size
- 🎵 **Sound Effects & Music** — full audio experience
- 🌈 **Neon Animated Border** — rainbow glowing game over screen

## 🕹️ How To Play

| Action | Control |
|--------|---------|
| Fly up | `Click` / `Tap` / `Space` / `↑` |
| Start game | Click **Play Now** button |
| Restart | Tap anywhere after Game Over |

### Power-ups
| Icon | Name | Effect |
|------|------|--------|
| 🛡 | Shield | Invincible — pipes can't hurt you |
| 🧲 | Magnet | Automatically pulls coins toward you |
| ⏱ | Slow-Mo | Slows everything down for 5 seconds |

## 🛠️ Run Locally
```bash
# Clone the repository
git clone https://github.com/yourusername/neon-orb.git

# Go into the folder
cd neon-orb

# Open in browser (just double click index.html)
# OR use Live Server in VS Code
```

> ⚠️ **Note:** Sounds require a local server to work.  
> Use **VS Code Live Server** extension or any local HTTP server.

## 📊 Difficulty Settings

| Setting | Value | Description |
|---------|-------|-------------|
| Gravity | 0.10 | Very floaty ball |
| Pipe Gap | 240px | Wide and forgiving |
| Start Speed | 1.5 | Slow and gentle |
| Max Speed | 3.2 | Never gets too fast |
| Speed Ramp | Every 20 pipes | Gradual increase |
